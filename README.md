# Hand History Whiteboard

A mobile-first poker hand recorder and visual replay tool. Record hands action-by-action using a horizontal scroll timeline, then review them on a structured whiteboard layout.

## Tech Stack

- **Frontend:** Vue 3 + Composition API + Capacitor (iOS/Android)
- **Engine:** `@robkerr1992/poker-engine` (client-side validation, betting, pots)
- **Backend:** Laravel API + MySQL
- **UI:** Tailwind CSS, mobile-optimized dark theme
